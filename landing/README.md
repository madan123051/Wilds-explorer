Production landing page for **Game of Wilds Explorer** deployed on Vercel.

👉 **https://wilds-explorer.vercel.app**

## 📦 What's Inside

- **index.html** - Full landing page with game vibe aesthetic
- **Nepal-inspired design** - Colors, themes, and visual style matching the game
- **Game features showcase** - Interactive feature cards
- **Documentation links** - Easy access to all dev docs
- **GitHub integration** - Direct links to source code

## 🎨 Design Features

✅ Nepal jungle color scheme (dark greens, gold accents)  
✅ Responsive design (mobile, tablet, desktop)  
✅ Game vibe aesthetic (not boring docs)  
✅ Fast loading (pure HTML/CSS)  
✅ Interactive elements (hover effects, animations)  
✅ Easy navigation  

## 🚀 Deployment

Automatically deployed to Vercel on every push to `main` branch.

```bash
# Local testing
open landing/index.html
```

## 📝 Updating the Landing Page

Edit `landing/index.html` and push to GitHub:

```bash
git add landing/index.html
git commit -m "[Landing] Update game vibe page"
git push origin main
```

Vercel will auto-deploy! 🚀
