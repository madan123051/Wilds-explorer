# Wilds Explorer - Unity Scripts

## Folder Structure

```
Scripts/
├── Core/
│   ├── Constants.cs       - Game constants
│   ├── GameManager.cs     - Main game controller
│   └── SaveSystem.cs      - Save/Load system
├── Player/
│   ├── PlayerController.cs - Player movement
│   ├── PlayerCamera.cs    - Camera follow
│   └── PlayerStats.cs     - Player statistics
├── Gameplay/
│   ├── CoinCollector.cs   - Coin collection
│   ├── MissionSystem.cs   - Mission management
│   └── LevelManager.cs    - Level control
└── UI/
    ├── UIManager.cs       - UI management
    └── HUD.cs            - Heads-up display
```

## Setup Instructions

1. Copy all files to Unity Assets/Scripts folder
2. No compilation errors should appear
3. Create scenes and assign scripts
4. Test in Unity Editor
